What Elaya is.
Elaya is a programming language and a command line compiler.
The goal of the project can be described with the following keywords:A little bit 
innovative, a little bit Traditional but not a clone, discover and improve.
Don't expect a fancy ide with a large RTL,it is just a command line compiler
with a small rtl. For feature and limitation here more
The language
The language has some similarities Pascal but has some extra interesting features


    

  •  Classes:

    • Nested classes

    • Virtual classes

    

  •  Routines:

    •  Extended Routines.

    •  Named Overloading, for more: type/name overloading.

    •  Named parameters

    

  •  Types:

    • Extra possibilties of the string type.

    

  • Autoloading Units.
